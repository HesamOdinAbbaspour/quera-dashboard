- 🎨 طراحی الهام‌گرفته از رابط کاربری سایت کوئرا

- 🔠 استفاده از فونت فارسی ایران‌یکان
- 💡 پیاده‌سازی فقط با HTML و CSS خالص
- 🧰 مناسب برای تمرین UI، طراحی Grid و نمونه‌کارهای ساده
